CRUD Top Menu


  1. Use the Top Menu tab to view list of Top menu items.
  2. View, Add, Edit, Delete Top Menu items.
  3. Activate / Deactivate to include / exclude from Portfolio.

CRUD Sub Menu


  1. Use the Sub Menu tab to view list of Sub menu items.
  2. View, Add, Edit, Delete Sub Menu items.
  3. Each Sub Menu will have parent Top Menu.
  4. Activate / Deactivate to include / exclude from Portfolio.

CRUD Pictures


  1. Use the Picture tab to view list of Pictures.
  2. View, Add, Edit, Delete Pictures.
  3. Each Picture will have parent Sub Menu.
  4. Activate / Deactivate to include / exclude from Portfolio.

Preview Portfolio


  1. Generates your Portfolio with all Active Top Menus, Sub Menus, Pictures.
